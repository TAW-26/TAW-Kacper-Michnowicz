# System rezerwacji boisk i obiektów sportowych

## 1. Opis projektu

Projekt umożliwia użytkownikowi zarządzanie rezerwacjami boisk oraz różnych obiektów sportowych, takich jak hale sportowe, korty tenisowe, boiska do piłki nożnej czy sale fitness. Celem systemu jest usprawnienie procesu rezerwacji zarówno dla użytkowników indywidualnych, jak i administratorów obiektu.

## 2. Instrukcja uruchomienia

### Backend

1. Przejdż do folderu backend
```
cd backend
```
2. Zainstaluj zależności
```
npm install
```
3. Przejdż do folderu src
```
cd src
```
4. Uruchom serwer poleceniem
```
node index.js
```

### Frontend

1. Przejdż do folderu frontend
```
cd frontend
```
2. Uruchom aplikację poleceniem
```
ng serve
```
3. Wejdź na adress http://localhost:4200
